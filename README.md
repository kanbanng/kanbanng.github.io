# OpenToDo

Open To-Do is an open source To-Do website that I made to learn more about web development.

## Main features
- Drag and drop using sortableJs
- JSON data import/export (+ drag and drop files)
- Card adding
- Auto-save progress to localStorage

## Known issues
- No adding columns
- No service worker
- Structured more like a desktop app than a web app

## How to use
Visit [https://open-todo.github.io](https://open-todo.github.io)
